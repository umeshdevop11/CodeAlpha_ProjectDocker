# Docker Web Server Project

## 🔹 Objective
Learn Docker containerization and deploy a web server.

## 🔹 Steps Performed
1. Installed Docker
2. Ran nginx container (port 8080)
3. Created custom HTML page
4. Mounted volume (port 8081)
5. Built Docker image (port 8082)
6. Used Docker Compose (port 8083)

## 🔹 Access URLs
http://<IP>:8080
http://<IP>:8081
http://<IP>:8082
http://<IP>:8083

## 🔹 Commands Used
- docker run
- docker build
- docker ps
- docker logs
- docker-compose up
